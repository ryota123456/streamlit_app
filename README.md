チーズ需給データ可視化アプリ

アプリ概要
e-Stat（政府統計）の「チーズの需給表」CSVデータを使って、チーズの生産量・輸入量・消費量・国産割合などの指標を、年次で可視化・比較できるStreamlitアプリ
サイドバーで指標と期間を選ぶと、表・グラフが自動で切り替わる

e-Statの情報
統計名：チーズの需給表（確報）  
表題：令和四年度のチーズの需給表 統計表  
データの取得日：2026-02-01

データの列の意味
このCSVは、指標（項目）ごとに年次（平成/令和）の値が並ぶ形式です。

チーズ需給表の項目名  ：国産ナチュラルチーズ生産量、輸入ナチュラルチーズ総量、チーズ総消費量、国産割合 など
年：年次
値：指標の数値

サイドバー
サイドバーで指標を選択できる
サイドバーで期間を選択できる
表とグラフを切り替えてデータを確認できる
解釈・考察タブで簡単な増減の説明を表示

タブ
目的：画面を「可視化 / データ確認 / 解釈・考察」に分けて見やすくする
使用箇所：メイン画面のタブ構成

整列
目的：列の説明を折りたたみ、画面をすっきりさせる
使用箇所：「データ確認」タブ内

ダウンロードボタン
 目的：絞り込み後のデータをCSVで保存できるようにする
 使用箇所：「可視化」タブ内
